# LWC storybook POC

Sample repository that uses storybook with Lightning Web Components

## Setup

> This project uses [Volta](https://volta.sh/) to manage Node.js and [yarn](https://yarnpkg.com/) versions.

Clone this repository

```sh
yarn install
```

## Run

```sh
yarn storybook
```

## Format

```sh
yarn prettier
```
